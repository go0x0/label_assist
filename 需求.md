# 系统功能概述  
一个图形界面程序，可以打开视频文件转换为图像帧，可以启动本地通过 uvx安装的 labelme。

# 功能点
## 视频转换功能
1、用户点击“选择视频“按钮，选择视频文件。界面上显示用户选择的视频文件路径。  
2、用户可以点击“选择输出目录“按钮，选择视频转为图像后的输出目录，界面上显示用户选择的输出目录，输出目录默认是视频文件同一文件夹下的以视频文件（去掉后缀名）为名称的目录。  
3、选择输出目录按钮右边有一个打开输出目录按钮，点击可以打开用户选择的输出目录。  
4、用户点击转换按钮，程序将用户选择的视频文件逐帧图像输出，图像命名规则为 img_%06.jpg，如果超过 999999 张，报错。转换前检查输出目录是否为空，非空则向用户确认是否继续转换。  
## 启动labelme功能
用户点击启动 labelme 按钮，尝试启动本地通过 uvx安装的 labelme。

# 技术要求
1、使用 python，在项目本地创建.venv，使用 pip 安装依赖库
2、打包成可以迁移到其他 macos 上使用的程序。提供可以供我手动执行的打包脚本。  
3、界面可以使用 pyside6，如果pyside6不利于满足上述要求 2，可以使用别的框架实现界面。  
4、视频转换成图像，要求逐帧转换，不要丢失图像帧，尽可能保持清晰度。
